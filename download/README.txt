# Universal AI Adapter - Quick Start

## Windows App

### Run It
1. Download `universal-ai-adapter.exe`
2. Double-click it
3. It starts the local server and opens `http://localhost:3000`

### Features
- Works offline with Ollama
- No installer required for the portable build
- Built-in web UI
- Compare multiple providers
- 15 provider configurations supported

### Requirements
- Windows 10/11
- Node.js for local server workflows
- Ollama from https://ollama.com if you want local free models

## Web

Open:
- `http://localhost:3000`

## NPM

```bash
npm install universal-ai-adapter
```

## Supported Providers

- Ollama: local, no key required
- LocalAI: local, no key required
- Cerebras: API key required
- OpenRouter: API key required
- Qwen: API key required
- Kimi: API key required
- Mistral: API key required
- Perplexity: API key required
- Minimax: API key required
- Z AI: API key required
- Gemini: API key required
- Groq: API key required
- DeepSeek: API key required
- OpenAI: API key required
- Anthropic: API key required
