Powerful Features

🔄

Auto Fallback

Automatically switch providers when one fails. Never lose your chat.

💾

Response Caching

Intelligent caching with LRU eviction. Save API calls and money.

⚡

Rate Limiting

Built-in rate limiting per provider. Avoid hitting API limits.

🔁

Retry Logic

Exponential backoff with jitter. Handles transient failures.

🔌

Circuit Breaker

Prevent cascading failures. Auto-recovery when services recover.

📡

Streaming

Real-time response streaming. See responses as they're generated.

🤖

Model Router

Auto-select best provider based on task. Coding, writing, analysis.

🐳

Free Local AI

Ollama integration. Run AI locally - free and private.

Simple to Use

// Install
npm install universal-ai-adapter

// Use in your code
import { UniversalAIAdapter } from 'universal-ai-adapter';

const adapter = new UniversalAIAdapter({
  provider: 'ollama',
  providers: {
    ollama: { baseURL: 'http://localhost:11434', model: 'llama3.2' }
  }
});

const response = await adapter.simpleChat('Hello!');
console.log(response);
