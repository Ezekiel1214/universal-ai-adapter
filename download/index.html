Simple to Use

// Install
npm install universal-ai-adapter

// Use in your code
import { UniversalAIAdapter } from 'universal-ai-adapter';

const adapter = new UniversalAIAdapter({
  provider: 'ollama',
  providers: {
    ollama: { baseURL: 'http://localhost:11434', model: 'llama3.2' }
  }
});

const response = await adapter.simpleChat('Hello!');
console.log(response);

Quick Setup Guide

🐳 Step 1: Install Ollama (Recommended - Free & Local)

  1. Download from ollama.ai
  2. Run the installer and follow prompts
  3. Open terminal and run: ollama pull llama3.2
  4. That's it! Ollama runs at http://localhost:11434

🖥️ Step 2: Run Universal AI Adapter

  1. Download the .exe from this page
  2. Double-click to run - it opens browser automatically!
  3. Or run node server.js in the folder
  4. Select "Ollama" provider and start chatting!

💡 Optional: Use API Keys

For OpenAI, Anthropic, Groq, or DeepSeek - get an API key from their websites and enter it in Settings (⚙️).

Pro tip: Groq and DeepSeek offer free API tiers!
